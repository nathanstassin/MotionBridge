## Overview
For detailed installation and usage instructions, visit the [project page](https://nathanstassin.com/motionbridge)

MotionBridge is a workflow automation tool that bridges the gap between Adobe After Effects and Blackmagic DaVinci Resolve. It enables motion designers and editors to work efficiently across both applications by synchronising compositions, markers, and renders, eliminating traditional export/import bottlenecks.
